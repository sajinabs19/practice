HTML

HTML (HyperText Markup Language)is a markup language .It contains different tags and attributes. HTML is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

HTML Tags are


  1. Heading
  2. Paragraph
  3. Anchortag
  4. Image
  5. Video
  6. Forms
  7. IFrame
  8. Tables
  9. Ordered List
  10. UnOrdered List
